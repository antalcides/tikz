\textbf{Objectifs }: 

\begin{itemize}

{\textbf{Objectives }: 

\begin{itemize}
\item One image per command or parameter.
\item the minimum amount of text possible.
\item the most complete possible update after update.
\item keep the same structure as VisualPSTricks
\end{itemize}}
